Basic Table Generator:

This is a basic table generator, that accepts an input for the rows and columns from the user that they want to generate after they press the "Generate" button. Pressing the "Reset" button will clear the table and reset the textboxes to the default value of one.

This uses a paragraph tag with no content and document.GetElementById to draw the table on the page.
